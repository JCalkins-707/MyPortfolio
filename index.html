Collaborative Projects

HCAR: The Studio - Art Management System

This project was for the Cal Poly Humboldt Hackathon for Social Good in Spring 2025. I collaborated with 3 other teammates to make a web application in approximately 20 hours. My primary role was database design and management. I created the database and ensured it was kept consistent between the model and the framework. We used the Django framework and MariaDB.

This project placed in the top 3 for the hackathon.

View the project on GitHub

3D Model Viewer

I worked within a development team to create a prototype web application that could accept, store, and render 3D model files. I was responsible for the UI/UX, the database design, and much of the documentation. We worked with SvelteKit, used the Drizzle ORM with SQLite, and used 3js to render the 3d models.

View the project on Github

Pet Cafe Database Prototype

This was a group assignment with two classmates and I. The goal was to create a prototype database, including a model and scenario, that could simulate something a real business might use. Our project featured PL/SQL triggers, formatted reports, 24+ distinct SQL queries, and an ERD, among other things.
